## About the Project

Roast & Petal is a high-converting single-page marketing website built for a specialty café brand. The design prioritises visual storytelling — warm typography, a structured content hierarchy, and layout patterns that guide the visitor from brand introduction through to reservation.

---

## What's on the Page

| Section | Description |
|---|---|
| **Navbar** | Sticky with smooth-scroll anchor links |
| **Hero** | Full-bleed image panel, headline, CTA buttons, floating rating badge |
| **Stats Bar** | 3-column milestone strip |
| **Menu** | Product cards with image, category, description, price, and add button |
| **Why Choose Us** | 3 icon-driven feature cards with hover accent animation |
| **Gallery** | Mosaic CSS Grid — first image spans two rows |
| **Testimonial** | Star rating + pull quote block |
| **Hours & Location** | Two-column contact info panel |
| **Footer** | Brand, copyright, social links |

## Getting Started

**Prerequisites:** Node.js 18+

```bash
# Clone the repo
git clone https://github.com/your-username/roast-and-petal.git
cd roast-and-petal

# Install dependencies
npm install

# Start dev server
npm run dev
```

App runs at `http://localhost:5173`

```bash
# Production build
npm run build
```
